# Claude Code プラグイン開発ルール

このディレクトリには、Claude Codeプラグイン開発時に参照すべきルールが格納されています。

## 📋 ファイル

- `README.md` - このファイル - ルールシステムの概要
- `output-language.md` - 出力言語ルール（日本語出力の原則）

## 🎯 使用目的

### ClaudeCodeへの指示

プラグイン開発時は、必ず以下のファイルを確認してください：

1. **output-language.md** - 出力言語ルール（最優先）

## 原則

- すべてのユーザー向け出力は日本語で行う
- コード要素（変数名、関数名など）は英語を使用
- ドキュメントとコメントは日本語で記述
