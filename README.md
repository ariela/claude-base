# claude-base

A Claude Code project with standard directory structure.

## Project Structure

```
claude-base/
├── .claude/              # Claude Code configuration
│   ├── commands/         # Custom slash commands
│   └── hooks/            # Session hooks
├── src/                  # Source code
├── tests/                # Test files
├── docs/                 # Documentation
├── .gitignore           # Git ignore rules
└── README.md            # This file
```

## Claude Code Features

### Custom Commands
Add custom slash commands in `.claude/commands/`. Each markdown file becomes a command.

### Hooks
Configure session hooks in `.claude/hooks/` to run scripts during Claude Code sessions.

## Getting Started

1. Open this project with Claude Code
2. Explore the directory structure
3. Add your source code in `src/`
4. Write tests in `tests/`
5. Create documentation in `docs/`

## Development

This project is set up for use with Claude Code, Anthropic's CLI for Claude.
